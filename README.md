# Face-Recognizer
A Basic Face Recognition App wrapped with the Flask micro-framework that is pre-trained for the faces of Narendra Modi and Arvind Kejriwal.

### Dependencies
  - OpenCV
  - OpenCV Contrib
  - NumPy
  - Flask
  - PyMongo
